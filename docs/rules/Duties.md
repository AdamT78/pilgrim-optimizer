# Duties (Current Scope)

Duty resolution compares active player acolytes at a selected duty against opponents:

- **majority**: value 2, silver cost 0
- **parity**: value 1, silver cost 0
- **minority**: value 1, silver cost 1

After resolving a duty, active player acolytes at that duty are recalled to the city.
Detailed duty definitions and effects are loaded from `configs/duties.json`.
